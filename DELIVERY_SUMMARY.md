## 📊 DATABASE STRUCTURE

Six MongoDB collections with proper schemas:

| Collection | Fields | Indexes |
|-----------|--------|---------|
| users | name, email, password, age, weight, height, etc. | email, userId |
| fitnesslogs | userId, date, type, duration, calories, etc. | userId, date |
| nutritionlogs | userId, date, mealType, calories, macros, etc. | userId, date |
| periodtrackers | userId, startDate, symptoms, mood, flow, etc. | userId, startDate |
| chatmessages | userId, role, content, timestamp | userId, timestamp |
| skinanalysises | userId, imageUrl, concerns, recommendations | userId, analysisDate |

All collections have proper indexing for optimal query performance.
